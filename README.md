###Expense Tracker
Автор: [Платон Мешалкин]
Язык: Python 3.x + Tkinter

###Описание программы
Expense Tracker — простое GUI-приложение для учёта личных расходов. Позволяет добавлять расходы с указанием суммы, категории и даты. Реализованы фильтрация по категории и дате, подсчёт суммы за выбранный период. Данные сохраняются и загружаются из файла JSON.

###Требования
Python 3.x (с модулем tkinter)
Опционально: установите зависимости через pip install -r requirements.txt
###Как использовать
Запустите файл expense_tracker.py.
Введите сумму (положительное число), категорию и дату (ГГГГ-ММ-ДД).
Нажмите «Добавить расход» — запись появится в таблице.
Для фильтрации выберите категорию или введите дату и нажмите «Применить фильтр».
Для подсчёта суммы за период введите даты начала и конца периода и нажмите «Сумма за период».
Для сохранения данных нажмите «Сохранить в JSON», для загрузки — «Загрузить из JSON».
Используйте Git для контроля версий.
## 5. Примеры использования

### Добавление расхода:
- Сумма: 350.50  
- Категория: Еда  
- Дата: 2026-04-23  
- Действие: «Добавить расход»
→ Запись появляется в таблице.

### Фильтрация по категории:
- Выбрать в фильтре: «Еда»
→ В таблице только расходы на еду.

### Подсчёт суммы за период:
- Начало: 2026-04-20  
- Конец: 2026-04-23  
→ Выводится сумма всех расходов за эти дни.

### Сохранение/загрузка:
- После добавления нескольких расходов нажать «Сохранить в JSON».
- Закрыть программу.
- Запустить снова и нажать «Загрузить из JSON» — данные восстановятся.
